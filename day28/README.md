# Day 28 - Pomodoro Timer (Tkinter GUI)

## Project Overview

On Day 28 of the 100 Days of Python Bootcamp, I built a **Pomodoro Timer** using Python's Tkinter library. This desktop application helps users improve productivity by following the Pomodoro Technique, which alternates focused work sessions with short and long breaks.

The application features a graphical interface with a countdown timer, automatic session switching, and progress tracking using check marks.

---

## Features

- 25-minute work sessions
- 5-minute short breaks
- 20-minute long break after every 4 work sessions
- Start timer functionality
- Reset timer functionality
- Automatic countdown updates
- Session labels (Work / Break)
- Visual progress using check marks
- Simple and user-friendly GUI

---

## Pomodoro Cycle

1. Work for 25 minutes
2. Take a 5-minute short break
3. Repeat the cycle 4 times
4. Take a 20-minute long break
5. Start the next cycle
