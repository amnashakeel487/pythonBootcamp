# 🏓 Pong Game (DAY22)

A classic Pong Game built using **Python**, **Object-Oriented Programming (OOP)**, and the **Turtle Graphics** library. This project is part of **Day 22** of the **100 Days of Python Bootcamp**.

---

## 📌 Features

- 🎮 Two-player gameplay
- ⌨️ Keyboard controls
- ⚡ Smooth ball movement
- 🏓 Ball bounces off paddles
- 🧱 Ball bounces off top and bottom walls
- 📊 Live score tracking
- 🔄 Ball resets after each point
- 🚀 Ball speed increases after paddle collisions
- 🏗️ Modular code using Object-Oriented Programming

## 🎮 How to Play

This is a **2-player game**.

### Left Player Controls

| Key | Action |
|-----|--------|
| W | Move Up |
| S | Move Down |

### Right Player Controls

| Key | Action |
|-----|--------|
| ↑ Up Arrow | Move Up |
| ↓ Down Arrow | Move Down |

---

## 🏆 Game Rules

- The ball starts from the center of the screen.
- Players use their paddles to hit the ball.
- The ball bounces off the top and bottom walls.
- If the ball hits a paddle, it bounces back.
- If a player misses the ball, the opponent earns one point.
- After each point, the ball returns to the center.
- The game continues until the players decide to close the window.
