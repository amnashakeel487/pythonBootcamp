# Day 27 - Tkinter, *args, **kwargs and GUI Programming

## Project Overview

On Day 27 of the 100 Days of Python Bootcamp, I learned the fundamentals of Graphical User Interface (GUI) development using Python's built-in Tkinter library. Instead of creating command-line programs, I built interactive desktop applications with buttons, labels, text fields, and event handling.

The main project for this day is a **Miles to Kilometers Converter**, where users can enter a distance in miles and instantly convert it to kilometers by clicking a button.

### Features
