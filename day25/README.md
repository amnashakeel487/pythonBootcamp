# Day 25 - Working with CSV Files and Analyzing Data with Pandas

## 📌 Overview

Day 25 focuses on learning how to work with CSV (Comma-Separated Values) files using Python and the Pandas library. This day introduces powerful data analysis techniques by working with DataFrames and Series, reading and writing CSV files, filtering data, performing calculations, and visualizing structured information.

To apply these concepts, a U.S. States Guessing Game was built using Turtle Graphics and Pandas. The game challenges users to identify all 50 U.S. states, displaying each correct answer on a map at its correct location.

## 📚 Projects Completed

### 1. Weather Data Analysis
- Read weather data from a CSV file.
- Calculated average and maximum temperatures.
- Filtered data based on specific conditions.
- Learned how Pandas simplifies CSV handling compared to Python's built-in csv module.

### 2. Squirrel Census Analysis
- Analyzed the Central Park Squirrel Census dataset.
- Counted squirrels based on fur color.
- Created a new DataFrame containing the analysis.
- Exported results to a new CSV file.

### 3. U.S. States Guessing Game
- Displayed a blank U.S. map using Turtle Graphics.
- Accepted state names from the user.
- Displayed correctly guessed states at their exact map coordinates.
- Tracked the player's score.
- Generated a `states_to_learn.csv` file containing all states not guessed by the player.
